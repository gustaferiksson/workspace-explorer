# Change Log

All notable changes to the "workspace-manager" extension will be documented in this file.

## [0.0.1] - 2026-01-11

### Added
- Initial release
- Repository browser for discovering Git repositories
- Configurable base paths for repository scanning
- Dynamic workspace folder views in Explorer
- Add repositories to workspace
- Remove repositories from workspace
- Open repositories in new window
- Stable view management (no reloading when adding/removing folders)
- Git-aware repository names from remotes
- Folder-first sorting in file trees
